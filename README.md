# JunMystery.github.io — Portfolio

**Live site**: [https://junmystery.github.io/](https://junmystery.github.io/)

Interactive portfolio of a Infrastructure Engineer & AI-Augmented Developer. Features a terminal-themed UI with easter eggs, CRT mode, mini-games, and a fully offline-friendly static build.

## Quick Start

Open **[junmystery.github.io](https://junmystery.github.io/)** in any modern browser. No install, no build step, no server needed.

## Development

Source files live under `src/bundle/` as ES5 modules. Rebuild with `pwsh src/bundle/build.ps1` or run the concatenation script manually.
